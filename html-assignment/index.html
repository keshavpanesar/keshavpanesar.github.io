Heading tag
This tag is used to make heading for our content. There are total of 6 heading going from
h1,h2... to h6. In which h1 is most important and biggest heading and h6 is for least important and
smallest heading.

Image tag
This is a singular tag to insert image in HTML file. It has 2 parts which are
src and alt. SRC is used to insert the path of image, and alt is used to give a breif
description about image for 'picture of keshavsinghpanesar'

Strong tag (bold)
This tag is used to make the text bold which highlighs the text to make it look important.
