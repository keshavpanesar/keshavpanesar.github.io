There are 5 tags which I learnt in the first month of my Web Development I.
Heading tag:
This tag is used to make heading for our content. There are total of 6 headings going from
h1,h2... to h6. In which h1 is most important and biggest heading and h6 is for least important and
smallest heading.
Image tag:
This is a singular tag used to insert image in HTML file. It has 2 parts which are
src and alt. SRC is used to insert the path of image, and alt is used to give a brief
description about image for 'picture of keshavsinghpanesar'
Strong tag:
This tag is used to make the text bold which highlights the text to make it look important.
Superscript and Subscript Text:
These tags are used to shift the words upwards (sup) and downwards (sub). For example: 10th and H2O.
HTML Anchor tag:
This tag is used to link other website or photos or documents on our web. For example: Click here to learn about me.
